### 1. stock-risk-control

基于2%风险原则的股票仓位管理工具，帮助你严格执行风险控制。

#### 核心概念

**2%风险原则**
- 任何一笔交易，最多只承担本金2%风险
- 亏损 = 仓位 × 止损跌幅
- 当价格跌到止损价时，亏损 = 2%

**两种模式**

| 模式 | 适用场景 | 仓位计算 |
|------|----------|----------|
| 2%集中 | 单品种趋势投资 | 2% / 止损跌幅 |
| 2%分散 | 多品种分散投资 | 每个品种独立计算2%风险 |

**模式自动切换**
- 仓位 ≤ 2% → 自动调整为"分散"
- 仓位 > 2% → 自动调整为"集中"

#### 使用示例

**计算仓位**
```
用户：/risk 集中 100 90
AI：📊 2%集中仓位计算
    现价：100
    止损价：90
    止损跌幅：10.00%
    建议仓位：20.0%
    ✅ 风险控制：最多承担本金2%风险
```

**查看持仓**
```
用户：/risk 列表
AI：📊 完整持仓股票列表

| 名称 | 仓位 | 盈亏% | 建议 | 现价 | 止损价 |
|------|------|-------|------|------|--------|
| 股票A | 8.0% | +5.20% | ⚠️ 降低到4.5% | 105.00 | 90.00 |
| 股票B | 5.5% | +2.10% | ⚠️ 降低到4.0% | 52.00 | 45.00 |
| 基金C | 4.0% | -1.50% | ✅ 持有 | 1.98 | 1.50 |
| 股票D | 3.2% | +8.00% | ✅ 持有 | 216.00 | 180.00 |
| ... |
```

**更新股价**
```
用户：/risk 更新 1 108.50
AI：✅ 股票已更新！ID: 1
    现价: 108.50
    盈亏: +8.50%
```

**查看整体盈亏**
```
用户：/risk 整体盈亏情况
AI：💰 总市值: 50,000.00 元
    💵 总成本: 48,000.00 元
    📈 总盈亏: +2,000.00 元 (+4.17%)
    ✅ 盈利股票: 8 只
    ❌ 亏损股票: 5 只
    📊 总仓位: 50.00%
```

#### 快捷指令

| 指令 | 说明 |
|------|------|
| `/risk 列表` | 查看持仓股票（默认不显示详情列） |
| `/risk 列表 显示ID` | 查看持仓（显示ID） |
| `/risk 列表 显示代码` | 查看持仓（显示股票代码） |
| `/risk 列表 显示理由` | 查看持仓（显示持有理由） |
| `/risk 列表 显示总值` | 查看持仓（显示总市值） |
| `/risk 添加 <名称> <代码> <模式> <数量> <成本> <止损> <现价>` | 添加股票 |
| `/risk 更新 <id> <现价>` | 更新股价 |
| `/risk 删除 <id>` | 删除股票（软删除） |
| `/risk 集中 <现价> <止损价>` | 计算2%集中仓位 |
| `/risk 分散 <现价> <止损价>` | 计算2%分散仓位 |
| `/risk 整体盈亏情况` | 查看总体盈亏 |

#### 列表显示规则

**默认展示列**：名称、仓位、盈亏%、建议、现价、止损价

**默认不展示列**（可通过参数开启）：
- ID（通过"显示ID"开启）
- 代码（通过"显示代码"开启）
- 成本价（通过"显示成本价"开启）
- 总值（通过"显示总值"开启）
- 模式列（通过"显示模式"开启）
- 持有理由（通过"显示理由"开启）

**盈亏符号规则**：
- 盈利：+5.2%
- 亏损：-3.1%
