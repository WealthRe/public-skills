# stock-risk-control

基于2%风险原则的股票仓位管理工具，帮助你严格执行风险控制。

## 核心概念

**2%风险原则**
- 任何一笔交易，最多只承担本金2%风险
- 亏损 = 仓位 × 止损跌幅
- 当价格跌到止损价时，亏损 = 2%

**两种模式**

| 模式 | 适用场景 | 仓位计算 |
|------|----------|----------|
| 2%集中 | 单品种趋势投资 | 2% / 止损跌幅 |
| 2%分散 | 多品种分散投资 | 每个品种独立计算2%风险 |

## 使用示例

### 计算仓位
```bash
/risk 集中 100 90
# 输出：
# 📊 2%集中仓位计算
# 止损跌幅：10.00%
# 建议仓位：20.0%
```

### 查看持仓
```bash
/risk 列表
# 输出表格：名称、仓位、盈亏%、建议、现价、止损价
```

### 更新股价
```bash
/risk 更新 1 105.50
```

## 快捷指令

| 指令 | 说明 |
|------|------|
| `/risk 列表` | 查看持仓股票 |
| `/risk 列表 显示代码` | 显示股票代码 |
| `/risk 添加 <名称> <代码> <模式> <数量> <成本> <止损> <现价>` | 添加股票 |
| `/risk 更新 <id> <现价>` | 更新股价 |
| `/risk 集中 <现价> <止损价>` | 计算2%集中仓位 |
| `/risk 分散 <现价> <止损价>` | 计算2%分散仓位 |

## 配置

- 数据库：SQLite3
- 默认总资金：100,000元（可自定义）
- 数据文件：`$DATA_DIR/stock_risk_control.db`

## 要求

- Python 3.x
- SQLite3
